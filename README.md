1. Double-click `index.html` or open it in your browser.
2. Alternatively, serve the folder with any static server.

### Optional: Simple Python server

```bash

# Then visit http://localhost:8080
```

## Stack
- Bootstrap 5 via CDN
- Google Fonts (Merriweather & Merriweather Sans)
- Custom theme styles in `assets/css/sarlis.css`
- Minimal interactions in `assets/js/main.js`
